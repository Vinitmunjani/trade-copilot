# SuperUI Skill

## Purpose
SuperUI is a specialized skill that helps design, analyze, and optimize user interfaces. It assists with layout structure, component architecture, styling systems, responsiveness, usability, and accessibility while ensuring modern design standards and clean frontend code.

---

## When To Use This Skill
Activate this skill when tasks involve:

- Building UI components
- Improving existing UI
- Fixing layout or alignment issues
- Making responsive designs
- Converting designs into code
- Styling with CSS / Tailwind / styled-components
- Refactoring frontend code
- Creating design systems
- Accessibility improvements
- Visual polish or UI audit

---

## Capabilities
This skill can:

### 1. UI Creation
- Generate complete UI components
- Build responsive layouts
- Create dashboards, forms, navbars, modals, tables
- Convert wireframes → code

### 2. UI Optimization
- Improve spacing, alignment, hierarchy
- Reduce visual clutter
- Enhance readability
- Improve color balance

### 3. Code Quality
- Refactor messy frontend code
- Apply best practices
- Ensure maintainability
- Enforce component reusability

### 4. Responsiveness
- Mobile-first layouts
- Adaptive breakpoints
- Fluid typography
- Grid optimization

### 5. Accessibility
- ARIA roles
- Semantic HTML
- Keyboard navigation
- Contrast compliance

---

## Design Principles Followed
SuperUI always prioritizes:

- clarity over decoration
- hierarchy over density
- consistency over creativity
- usability over aesthetics
- performance over heavy visuals

---

## Output Style Rules
When generating UI:

- Use modern design patterns
- Prefer reusable components
- Avoid inline styles unless necessary
- Use consistent spacing scale
- Follow naming conventions
- Keep code clean and structured

---

## Framework Preferences
Default stack assumptions unless specified:

- React + Tailwind
- Semantic HTML
- CSS Flex/Grid
- Component-based architecture

If the user specifies a framework, adapt instantly.

---

## Example Tasks

**User:** Create a responsive pricing section  
**Skill Behavior:** Generates structured responsive component with clean layout and consistent spacing.

**User:** Fix alignment issue in navbar  
**Skill Behavior:** Analyzes layout and returns corrected code with explanation.

**User:** Improve this UI design  
**Skill Behavior:** Reviews UI and suggests visual + structural improvements.

---

## Constraints
- Do not add unnecessary animations unless requested
- Avoid heavy libraries unless required
- Keep performance optimized
- Follow minimalistic modern UI standards

- "modern UI" → apply latest design trends
- "premium look" → enhance typography + spacing + shadows
- "minimal" → reduce visual noise and elements
